## Descrição 📄

O Pokédex é um aplicativo Android desenvolvido como um estudo de caso de MVVM, utilizando a PokeAPI para criar uma Pokédex. A aplicação permite aos usuários visualizar informações sobre diferentes pokémons, como habilidades, tipos, estatísticas e muito mais.

## Recursos Principais ⭐️

- 📚 Utilização da PokeAPI: O aplicativo consome a PokeAPI, uma API pública que fornece dados completos sobre os pokémons. Ele utiliza o Retrofit2 para realizar a análise de JSON e obter informações atualizadas sobre os pokémons.

- 🖼️ Carregamento Assíncrono de Imagens: O aplicativo utiliza a biblioteca Accompanist-Coil para carregar as imagens dos pokémons de forma assíncrona. Isso garante uma experiência de usuário suave e evita bloqueios durante o carregamento das imagens.

- 🪓 Logs com Timber: O Timber é utilizado para registrar logs durante o desenvolvimento do aplicativo, facilitando a depuração e o acompanhamento de problemas.

- ⏱️ Chamadas Assíncronas com Coroutines: O aplicativo utiliza coroutines do Kotlin para realizar chamadas assíncronas, garantindo um desempenho eficiente e uma experiência fluida para o usuário.

- 🔧 Injeção de Dependência com Dagger-Hilt: O Dagger-Hilt é utilizado para fornecer injeção de dependência no aplicativo, facilitando a organização e o gerenciamento de componentes e dependências.

## Uso 🚀

1. Instale o Android Studio.
2. Clone este repositório para sua máquina local.
3. Abra o projeto no Android Studio.
4. Aguarde a sincronização do projeto e a instalação das dependências.
5. Execute o aplicativo em um dispositivo Android ou emulador.
